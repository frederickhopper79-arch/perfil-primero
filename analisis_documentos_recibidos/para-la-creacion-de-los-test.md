Para la creación de los Test

La opción 100% Gratuita: Desarrollarlo tú mismo (Código Abierto)

Si quieres coste cero sin importar cuántos candidatos apliquen en tu web, debes programar las pruebas localmente en tu base de datos:

Para el Test de Personalidad (Gratis): El modelo Big Five (OCEAN) y el test DISC son de dominio público. Existe un proyecto open-source muy popular llamado BigFive-Test en GitHub que te permite ver cómo estructurar las preguntas de forma gratuita. Puedes replicar el cuestionario de preguntas en tu base de datos y calcular los resultados con algoritmos matemáticos simples sin pagar APIs externas.

Para el Test de Inglés (Gratis): Puedes estructurar un banco de preguntas propio de opción múltiple (gramática, vocabulario y comprensión lectora) dividido por niveles (A1 a C2). Al responderse dentro de tu sitio web, la corrección es automática, instantánea y totalmente gratuita.
