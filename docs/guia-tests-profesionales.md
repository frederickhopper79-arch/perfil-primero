# Guia para profesionalizar tests - Perfil Primero

## Decision recomendada

Los tests internos deben mantenerse como filtros orientativos hasta que exista validacion profesional. No deben llamarse "psicometricos certificados" ni usarse como unico criterio de seleccion.

## Ingles

Usar el Marco Comun Europeo de Referencia (CEFR/MCER) como escala de interpretacion: A1, A2, B1, B2, C1, C2.

Implementacion sugerida:

1. Separar lectura, vocabulario laboral y comprension de instrucciones.
2. Mostrar resultado como nivel estimado, no certificacion.
3. Permitir que el postulante suba certificado externo si tiene EF SET, Cambridge, IELTS, TOEFL u otro.
4. Mantener disclaimer: "evaluacion interna no certificada".

Fuentes consultadas:

- Council of Europe / CEFR Companion Volume: https://www.coe.int/en/web/common-european-framework-reference-languages
- EF SET como referencia gratuita de evaluacion online: https://www.efset.org/
- Cambridge/Oxford como certificaciones externas reconocidas: https://www.cambridgeenglish.org/ y https://elt.oup.com/feature/global/oxford-test-of-english/

## Personalidad / conducta laboral

Usar Big Five como marco base solo para senales generales de estilo laboral. Para items publicos, revisar IPIP, que es un banco de items de dominio publico asociado a investigaciones sobre Big Five.

Implementacion sugerida:

1. No entregar diagnosticos psicologicos.
2. Reportar dimensiones en lenguaje laboral: orden, colaboracion, energia social, apertura al aprendizaje, manejo de presion.
3. Evitar decisiones automaticas de rechazo.
4. Registrar consentimiento del postulante.
5. Agregar revision de psicologo laboral antes de usar resultados comercialmente.

Fuentes consultadas:

- International Personality Item Pool (IPIP), Oregon Research Institute: https://ipip.ori.org/
- Big Five personality traits como marco ampliamente usado en investigacion.
- Literatura reciente sobre uso de IA en evaluacion de personalidad advierte que requiere calibracion y validacion.

## Espanol profesional

No existe un unico estandar universal equivalente al CEFR para espanol laboral chileno en seleccion general. Se recomienda construir prueba funcional:

1. Comprension de instrucciones.
2. Redaccion breve de correo laboral.
3. Ortografia y puntuacion aplicada.
4. Tono profesional ante cliente o jefatura.

Resultado sugerido: "Basico", "Funcional", "Bueno", "Avanzado", nunca "certificado".
