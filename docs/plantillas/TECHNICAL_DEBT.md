## Cómo Registrar un Nuevo Ítem

1. Asignar el siguiente ID secuencial disponible.
2. Describir el problema concreto, no una queja genérica ("el módulo X no
   valida entradas duplicadas" en vez de "el módulo X está mal hecho").
3. Estimar el riesgo de no resolverlo y la prioridad relativa frente a otras
   deudas ya registradas.
4. Vincular, si corresponde, al Roadmap o a una RFC.

Toda deuda técnica que condicione una decisión del Roadmap deberá
reflejarse también en `ROADMAP.md` (HTES Cap. 20 — Gestión del Ciclo de
Vida del Producto).
