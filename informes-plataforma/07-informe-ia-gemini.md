# Informe IA Gemini

## Usos implementados

- Análisis de CV.
- Generación de perfil profesional.
- Mejora de perfil.
- Asistencia de matching para empresas.

## Control de errores

Si Gemini no responde por cuota o clave faltante, la UI no debe mostrar JSON técnico al usuario. El sistema usa fallback editable para mantener continuidad operacional.

## Riesgo principal

La IA no corrige una vacante mal escrita ni reemplaza criterio humano. Debe apoyar selección, no decidir contratación por sí sola.
